# Sistema Web de Gestión de Ventas

Trabajo Final Integrador
Tecnicatura Universitaria en Programación
Universidad Tecnológica Nacional - Facultad Regional Rafaela

## Descripción

Sistema web desarrollado para la administración de ventas en pequeños comercios.

Permite gestionar:

- Autenticación de usuarios
- Gestión de productos
- Gestión de categorías
- Registro de ventas
- Carrito de compras
- Historial de ventas
- Dashboard con estadísticas
- Control automático de stock

## Tecnologías utilizadas

### Backend
- NestJS
- TypeScript
- TypeORM
- MySQL

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap 5
- Chart.js
